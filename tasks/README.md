# Tasks Folder Naming Standard

Use this filename format for all items in this folder:

`NNN-type-kebab-case.ext`

Where:

- `NNN` = 3-digit sequence (`001`, `002`, ...)
- `type` = one of `task`, `issue`, `prd`
- `kebab-case` = short descriptive slug
- `ext` = `.md` or `.json` as needed

## Numbering Rules

- Use one global sequence across all types.
- Pick the next available number when adding a new file.
- Do not renumber existing files unless doing an explicit reorganization pass.
